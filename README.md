## Inspiration Interfaces
List the inspiration interfaces you used and briefly explain what you learned from them.
A portfolio for a digital marketing student. The first thing I noticed about the design was the color blocking and color palette. I learned that the color blocking can break up the webpage into different areas. I also was inspired by when you click it relocates you down the page. The other interface was a student homepage for Alyssa Jennings. It features a large background photo, and as you scroll, the center section containing her information moves down the page. I like that she has several navigation links at the top, making the website easy to navigate. 

## Design Choices
- Layout: Color blocking, with a background you can scroll over. I used flex boxes to help align each section
- Color: Pastel with darker text color
- Typography: Sans-Serif in different sizes
- Visual hierarchy: The first section has the biggest text, with the color blocking it help draw your attention to the top first. 
- Feedback states: The text changes color when you hover on the text in the navigation bar, and it brings your to a different part of the page. The GitHub button changes color and will bring you to the website. 
- Responsive design: The layout changes for a small screen. The Introduction section and about me sections words stack. The “my skills” circles come together and stack.
